# Contributing to Nexus Axiom

Thank you for your interest in contributing! This project aims to make eBPF security accessible.

## Quick Start

1. **Fork the repo**
2. **Clone your fork**
3. **Create a branch:** `git checkout -b feature/your-feature`
4. **Make changes**
5. **Test:** `cargo test && cargo clippy`
6. **Commit:** `git commit -m "feat: your feature"`
7. **Push:** `git push origin feature/your-feature`
8. **Open PR**

## Good First Issues

Look for issues labeled `good first issue` - these are perfect for new contributors.

## Areas We Need Help

- **Testing:** More CVE test cases
- **Documentation:** Improve installation guides
- **Performance:** Benchmark optimizations
- **Features:** Additional eBPF hooks
- **Integrations:** SIEM connectors

## Code Style

- Run `cargo fmt` before committing
- Run `cargo clippy` and fix warnings
- Add tests for new features
- Update README if adding features

## Questions?

Open an issue or discussion - we're friendly!
